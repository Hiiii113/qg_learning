# AI使用与个人贡献说明

## 一、AI 完成部分

- 使用 AI 辅助编写前端页面，实现前端逻辑
- 使用 AI 编写 CSS 样式，美化 ui 和交互逻辑

## 二、自行修改部分

- 撰写`request.js`用于封装 axios 并统一提交 Token
- 调整 AI 调用的接口相关问题
- 优化不同位置的跳转体验和提示信息

## 三、遇到的问题及解决

1. 增加图片上传功能时，前端的图片预览效果不理想
   - 查看博客找到了`URL.createObjectURL`这个方法生成临时 URL解决了
2. 重复选择同一图片时，`@change`事件无法触发
   - 查看博客找到了`e.target.value = ''`这个方法
