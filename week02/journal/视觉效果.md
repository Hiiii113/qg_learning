# 视觉效果

在前端页面设计中，为了给予良好的视觉效果，通常需要让鼠标滑动和鼠标点击变得可视化

## 一、按钮选中

当我们需要设计一些导航栏的时候，需要给选中的按钮加一个 CSS 样式用于区分选中和未选中

```java
<button :class="{ active: currentView === 'bind' }" @click="currentView = 'bind'">
    绑定宿舍
</button>
```

```css
button.active {
    background: #409eff;
    color: #fff;
    border-color: #409eff;
}
```

> 这里的`:class`是 v-bind 的简化写法，目的是给当前按钮的 class 绑定一个动态值：`{ active: currentView === 'bind' }`
>
> - 当 `currentView` 是 `bind`的时候，当前按钮拥有了`active`这个 class，CSS 样式起效让按钮加上**选中**的效果
> - 当 `currentView` 不是 `bind`的时候，当前按钮失去`active`这个 class，CSS 样式失效
